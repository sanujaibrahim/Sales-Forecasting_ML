# Sales-Forecasting_ML
# Sales Forecasting Using Machine Learning

## Overview
This project predicts future sales using historical sales data and machine learning.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Machine Learning Model
- Random Forest Regressor

## Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Sales Prediction

## Results
- Predicted future sales using historical data.
- Evaluated the model using MAE and RMSE.
